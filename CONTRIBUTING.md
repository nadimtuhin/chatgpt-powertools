# Contributing to ChatGPT Powertools

Thanks for taking the time to contribute!

## Getting Started

1. Fork the repo and clone it locally
2. Install dependencies: `yarn install`
3. Start dev build: `yarn dev`
4. Load the `dist/` folder as an unpacked Chrome extension (Developer Mode → Load unpacked)
5. Open [chat.openai.com](https://chat.openai.com) to test

## Pull Requests

- Keep PRs focused — one feature or fix per PR
- Run `yarn build` and verify the extension loads before submitting
- Describe what changed and why in the PR description

## Reporting Bugs

Open an issue with:
- Chrome version
- Steps to reproduce
- Expected vs actual behaviour
- Screenshot if applicable

## Code Style

This project uses Prettier. Run `npx prettier --write .` before committing.
